# 经典案例与实战技巧

## 一、经典格局案例

### 案例1：飞鸟跌穴 — 求财必得

**盘面**：阳遁3局，丙加戊落震三宫，临生门、六合
**求测**：一笔投资能否获利？

**分析**：
- 飞鸟跌穴（大吉格）+ 生门（财门）+ 六合（和合）
- 震三宫属木，春季木旺，格力量强
- 值符落乾六宫临天心（大吉），主有核心人物支持

**判断**：🟢 **大吉**，投资可获利的概率很高
**应期**：震三宫 → 3天内/3个月内见分晓（值符临九天加速，可能3天内）

**验证要点**：若同时逢庚在对面兑七宫冲克，则利润可能打折扣。

---

### 案例2：青龙逃走 — 破财之兆

**盘面**：阴遁6局，乙加辛落离九宫，临开门、白虎
**求测**：最近财运如何？

**分析**：
- 青龙逃走（大凶格）：乙奇被辛金冲克
- 离九宫属火，但辛金坐火地 = "金被火炼" = 财受损
- 开门虽吉但逢白虎 = 事业上有破财之象
- 值符落坎一宫逢腾蛇 = 主导之事缠绕不清

**判断**：🔴 **近期防破财**，尤其警惕：
- 合同/文书方面的损失（乙=文书）
- 被人欺骗或背叛（辛=错误/过失）
- 西方/西北方的人或事

**破局**：
1. 近期不签重要合同
2. 已签的合同仔细审查条款
3. 避免借钱给他人
4. 若已发生损失，找天心星所在方向的专业人士帮忙

---

### 案例3：伏宫格 — 大祸将至

**盘面**：阳遁7局，庚加戊落坤二宫，临死门、腾蛇
**求测**：公司最近是否安全？

**分析**：
- 伏宫格（大凶格）：庚金克戊土，大祸之象
- 死门 + 坤二宫（死门本宫）= 凶上加凶
- 腾蛇 = 怪异缠绕之事
- 但！坤二宫旬中若逢空亡 → 凶事落空

**判断**：
- **不空亡**：🔴 大凶，近期必有重大变故（官司/事故/重大损失）
- **若空亡**：🟡 凶事不成，但虚惊一场

**破局**（若不空亡）：
1. ⏸️ 立即暂停一切重大决策和扩张
2. 🔍 全面排查内部风险（财务/法律/安全）
3. 👨‍⚕️ 聘请专业人士（律师/顾问）预审
4. 📍 避免西南方向的事务
5. 📅 至少静观一个节气（15天）

### 案例4：神遁 — 如有神助

**盘面**：阳遁1局，生门+丙奇+九天同落巽四宫
**求测**：这个项目能成吗？

**分析**：
- **神遁格**（大吉）：生门+丙+九天三美汇聚
- 巽四宫属木，春末夏初木有余气
- 九天 = 进取飞扬之神
- 生门 = 财源生机之门
- 丙奇 = 月奇，逆乱中见转机

**判断**：🟢 **大吉**，如有神助，谋为皆利
**建议**：立即行动，不要犹豫。此格时效性强，宜在当前节气内推进。

### 案例5：吉凶参半 — 如何取舍？

**盘面**：
- 值符落乾六宫：刑格遇格（大凶）❌
- 用神宫（开门）落坎一宫：奇仪相佐（吉）✅
- 生门落巽四宫：飞鸟跌穴（大吉）✅
- 但艮八宫：腾蛇相缠+蛇相冲（双凶）❌

**求测**：要不要跳槽？

**分析**：
- **事业（开门）**：吉 → 新机会本身不错
- **但领导/核心环境（值符）**：大凶 → 新公司的上级或平台有大问题
- **财运（生门）**：大吉 → 薪酬待遇很好
- **但内部关系（艮八宫）**：双凶 → 同事关系或内部政治复杂

**判断**：🟠 **钱多但水深**
**破局方案**：
1. 如果只看钱 → 可以去，但做好"斗争"准备
2. 如果求稳 → 不建议，值符宫大凶意味着根本性问题
3. **折中方案**→ 去可以，但：
   - 不长期绑定（ contract / 短期优先）
   - 搞清楚汇报关系（避开值符宫所代表的方向/人）
   - 薪谈好再入场（利用生门大吉）
   - 不介入内部斗争（避开艮八宫代表的东北方势力）

## 二、实战技巧

### 技巧1：快速扫盘法（30秒定大局）

新手拿到一个盘面后，按以下顺序快速扫描：

```
① 看局名 → 阳遁还是阴遁？几局？
    ↓
② 找值符 → 落哪宫？逢什么？（3秒判断全盘基调）
    ↓
③ 数吉凶 → 快速数一下吉格vs凶格各几个（10秒）
    ↓
④ 定用神 → 根据问题找到用神宫（5秒）
    ↓
⑤ 看用神 → 空？墓？格？（5秒）
    ↓
⑥ 下结论 → 守/攻/观？（2秒）
```

### 技巧2："一宫断事"精简法

对于简单问题，只需看一个宫位：

| 问题 | 只看这一宫就够了 |
|------|------------------|
| "这件事能成吗？" | 值符宫 |
| "能赚多少钱？" | 生门宫 |
| "他对我怎么样？" | 休门宫（+乙庚关系）|
| "身体有没有问题？" | 天芮宫 |
| "考试行不行？" | 天辅宫 |
| "会不会打官司？" | 惊门宫 |
| "方向怎么选？" | 值符宫方位 |

### 技巧3：对比分析法

同一件事在不同时间起局，对比两个盘面：

| 对比项 | 含义 |
|--------|------|
| 局数变化 | 大环境能量场转变 |
| 值符宫移动 | 主导力量的方向变化 |
| 用神宫从凶→吉 | 时机转好，可以行动 |
| 用神宫从吉→凶 | 时机已过，不宜强行 |
| 格局增多/减少 | 局势在明朗化/复杂化 |

### 技巧4：五行追踪法

追踪某个五行的所有出现位置：

```
例：问财运，追踪"土"（生门属土、戊属土）
├── 戊在哪宫？→ 资本在哪里
├── 生门在哪宫？→ 财路在哪里  
├── 土宫是否旺相？→ 财运强弱
├── 土被什么克？→ 财运的威胁
└── 土生什么？→ 财的去向
```

### 技巧5：反证法

当你觉得盘面显示"吉"但直觉不对时：

1. **检查空亡**：是不是吉格落空亡了？
2. **检查入墓**：三奇是不是入墓了？
3. **检查击刑**：六仪是不是逢击刑了？
4. **检查伏吟**：全盘是不是伏吟了？
5. **检查时辰**：起局时间对不对？（差一个时辰完全不同）

> **直觉往往是潜意识捕捉到了你忽略的细节。**

## 三、常见问答模板

### Q：奇门准不准？
A：奇门遁甲是一个**趋势分析工具**，不是水晶球。它的价值在于：
- 帮你看到局势的全貌（四层盘面）
- 发现潜在的机会和风险（格局识别）
- 提供应对策略（破局方法论）
准不准取决于：起局时间是否精确、解盘是否全面、判断是否理性。

### Q：同一件事能不能再排一次？
A：原则上**一事不二占**。若第一次结果不满意：
- 先反思问题是否够明确
- 等至少一个时辰（4小时）
- 换一个角度提问（不是重复同一个问题）

### Q：奇门和其他术数（如紫微、六爻）矛盾怎么办？
A：不同术数看问题的角度不同：
- **紫微斗数**看人的本质特征和终身格局
- **六爻**看具体事件的发展过程
- **奇门遁甲**看当前时空下的最佳策略
三者互补而非矛盾。就像：性格（紫微）+ 过程（六爻）+ 决策（奇门）。

### Q：阴遁是不是就不好？
A：不是。阴阳遁只是能量的运动方式不同：
- **阳遁**：气在上升扩散 → 利于开创、进取、外放之事
- **阴遁**：气在下降收敛 → 利于固守、隐秘、内收之事
问防守、储蓄、地下工作、隐秘行动，阴遁反而更佳。

### Q：需要提供出生时间吗？
A：奇门是**时空预测术**，不需要出生时间。它看的是**"提问的那一刻"**的宇宙能量场分布。所以：
- 时辰越精确越好（精确到几点几分最理想）
- "现在"起局 = 看当前的态势
- "未来某时"起局 = 看那个时刻的态势

## 四、学习进阶路径

```
入门（1-2周）
├── 记住九宫八卦方位
├── 记住八门吉凶和基本含义
├── 理解三奇六仪是什么
└── 能看懂基本排盘输出
    ↓
进阶（1-2月）
├── 背诵20个最常见格局
├── 掌握用神取用法
├── 学会吉凶加权评分
└── 能独立完成基础解盘
    ↓
精通（3-6月）
├── 熟悉全部80+格局
├── 掌握应期判定
├── 灵活运用五大破局策略
└── 能处理复杂的复合问题
    ↓
大师（1年+）
├── 形成自己的解盘风格
├── 能结合其他术数交叉验证
├── 对特殊格局有独到见解
└── "心中无盘，处处是盘"
```
