# 吉凶判断方法论

## 一、吉凶判断的层次体系

奇门遁甲的吉凶判断不是非黑即白的，而是一个**多层次、多维度**的综合系统：

```
第一层：单宫吉凶（基础）
    ↓
第二层：格局吉凶（核心）
    ↓
第三层：用神关系（应用）
    ↓
第四层：全盘综合（总体）
    ↓
第五层：应期判断（时机）
```

## 二、五步判断法

### Step 1: 定大局 — 看局数与阴阳遁

| 类型 | 特征 | 基调 |
|------|------|------|
| 阳遁1-3局 | 初生之气，力量渐长 | 利于开创、起步 |
| 阳遁4-5局 | 盛极将转 | 利于收获、巩固 |
| 阴遁6-7局 | 阴气初盛 | 宜守不宜攻 |
| 阴遁8-9局 | 阴盛阳衰 | 大宜固守，忌妄动 |

**口诀**：阳遁利进阴遁守，初局宜创末局收。

### Step 2: 找值符 — 全盘统领

**值符是奇门盘面的"总指挥"，其落宫状态决定全盘基调：**

| 值符落宫状态 | 含义 | 应对策略 |
|-------------|------|----------|
| 临三奇+吉门+吉神 | 天时地利人和 | 积极进取，把握机会 |
| 临吉门但逢凶格 | 有机遇但有阻碍 | 小心行事，步步为营 |
| 临凶门+凶格 | 主导之力受阻 | 以守为主，等待时机 |
| 落空亡 | 指挥失灵，群龙无首 | 不宜有大动作 |
| 临庚（阻隔） | 最大障碍在核心位置 | 必须先排除内部阻力 |

### Step 3: 宫位加权评分

每个宫位按以下维度打分（每项±2分）：

#### 维度A：八门（权重30%）
| 门 | 分数 | 说明 |
|----|------|------|
| 生门 | +2 | 最吉之门 |
| 开门 | +1.5 | 吉门 |
| 休门 | +1.5 | 吉门 |
| 景门 | 0 | 平门（可视情况±0.5）|
| 杜门 | -0.5 | 平偏凶 |
| 伤门 | -1.5 | 凶门 |
| 惊门 | -1.5 | 凶门 |
| 死门 | -2 | 最凶之门 |

#### 维度B：九星（权重20%）
| 星 | 分数 | 说明 |
|----|------|------|
| 天心 | +2 | 大吉 |
| 天辅 | +2 | 大吉 |
| 天禽 | +1 | 吉 |
| 天任 | +1 | 小吉 |
| 天冲 | +1 | 小吉 |
| 天英 | 0 | 平 |
| 天芮 | -2 | 凶（病星）|
| 天蓬 | -1.5 | 凶（暗助）|
| 天柱 | -1 | 凶/小吉 |

#### 维度C：八神（权重20%）
| 神 | 分数 | 说明 |
|----|------|------|
| 值符 | +2 | 最贵之神 |
| 九天 | +1 | 进取 |
| 九地 | +1 | 固守 |
| 六合 | +1.5 | 和合 |
| 太阴 | +1 | 隐吉 |
| 白虎 | -1.5 | 凶 |
| 玄武 | -1.5 | 半凶 |
| 腾蛇 | -1 | 半凶 |

#### 维度D：格局（权重30%）
直接使用格局库中的等级：
- 大吉：+3
- 吉/小吉：+2
- 平/平吉：0
- 平凶：-1
- 凶/半凶：-2
- 大凶：-3

**注意**：若一宫同时有多个格局，取绝对值最大的一个（极端优先原则）。

### Step 4: 用神定位法

根据问题类型找到对应**用神宫位**后，对该宫进行重点分析：

```
用神宫位分析清单：
├── 1. 该宫总评分是多少？
├── 2. 是否逢空亡？（空亡则减半或作"虚无"论）
├── 3. 三奇是否入墓？（入墓力量大减）
├── 4. 与值符宫的五行关系如何？
│   ├── 值符生用神 → 外部助力强 ✅
│   ├── 用神生值符 → 我去助人，消耗 ⚠️
│   ├── 值符克用神 → 外来压力 ❌
│   └── 用神克值符 → 我能掌控 ✅
├── 5. 三方四正是否有救应？
│   ├── 三合宫有吉门吉格 → 有救 ✅
│   └── 对宫或三合宫逢凶 → 雪上加霜 ❌
└── 6. 旬中首尾是否相关？
    └── 值使门是否临用神宫？→ 事必至
```

### Step 5: 综合判定公式

```
总体吉凶 = Σ(各宫加权分) / 有效宫数 × 大局系数

其中：
- 大局系数：阳遁=1.0~1.2, 阴遁=0.8~1.0
- 若值符宫见大凶格，总分×0.7
- 若有三奇得使/神遁等大吉格，总分×1.3
- 若全盘伏吟，总分×0.5（大事不宜）
```

**评定标准：**
| 总分 | 判定 | 策略 |
|------|------|------|
| ≥+1.5 | 🟢 大吉 | 积极进取，大胆行动 |
| +0.5~+1.4 | 🟢 吉 | 可以行动，把握机会 |
| 0~+0.4 | 🟡 小吉/平 | 稳中求进，可试可行 |
| -0.4~-0.1 | 🟠 小凶/平 | 谨慎为宜，不可冒进 |
| -0.5~-1.4 | 🟠 凶 | 固守本位，以静制动 |
| ≤-1.5 | 🔴 大凶 | 停止一切重大行动 |

## 三、特殊情况的吉凶辩证

### 凶中藏吉
- **空亡逢凶**：凶格遇空亡 = 凶事不成 = 反吉
  - 例：伏宫格（大凶）若落空亡 = "大祸"落空 = 反而无祸
- **入墓之凶**：入墓虽凶，但也代表"被收藏"
  - 例：财星入墓 = 钱被藏起来了（还在，只是暂时拿不出）

### 吉中藏凶
- **吉格逢冲**：再好的格局被对宫冲克也减力
- **三奇遇庚**：最吉的三奇遇到最凶的庚 = 吉被破坏
- **吉门落死地**：开门落死门之宫 = "开启"变"终结"

### 吉凶转换
- **伏吟→反吟**：长期压抑后的突然爆发
- **填实空亡**：空亡被冲实之时 = 原本"没有"的事突然出现
- **冲起入库**：入墓被冲 = 原本"藏着"的东西出来了

## 四、常见误区

### ❌ 误区1："吉格多就一定吉"
- 一个大凶格可以抵消多个小吉格
- 格局有**质量差异**，不是数量游戏

### ❌ 误区2："用神宫吉就一定成"
- 还要看用神是否空亡、入墓、被克
- 全盘环境也很重要

### ❌ 误区3："阴遁就一定不好"
- 阴遁利于：固守、收藏、隐秘之事
- 问防守、储蓄、隐秘行动，阴遁反而更佳

### ❌ 误区4："值符落哪宫就往哪方向走"
- 值符是"主导力量"的方向
- 但具体还要看用神宫的位置和状态

### ✅ 正确思路
> **全局观 > 局部观 | 质量 > 数量 | 辩证看待 > 机械套用**
