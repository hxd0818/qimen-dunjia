# 奇门遁甲排盘解卦系统

> ⚔️ 中国古代最高级术数——奇门遁甲 Agent Skill
> 
> 支持时家奇门完整排盘、格局识别、吉凶判断与破局建议

## 功能特性

- ⚔️ **时家奇门完整排盘** — 四层盘面（地盘/天盘/人盘/神盘）九宫全排列
- 🔍 **80+ 格局自动识别** — 大吉格、吉格、大凶格、凶格全覆盖
- ☯️ **三遁八诈** — 天遁/地遁/神遁/风遁/龙遁/虎遁等特殊格局
- ⭕ **空亡判断** — 旬中空亡自动计算与影响分析
- 📊 **总体评估** — 吉凶量化打分 + 总体判断
- 💡 **破局建议** — 根据盘面自动生成应对策略
- 🎯 **用神系统** — 事业/财运/感情/健康/考试/官讼等15+场景
- 📋 **JSON输出** — 支持程序化二次分析

## 快速使用

```bash
# 当前时间起局
python3 scripts/qimen.py -q "最近事业运如何"

# 指定时间 + JSON输出
python3 scripts/qimen.py -y 2026 -m 5 -d 1 -H 9 --json
```

## 核心概念

### 四层盘面
1. **地盘** — 三奇六仪固定位置（基础能量场）
2. **天盘** — 三奇六仪随值符转动（动态能量）
3. **人盘** — 八门（开休生伤杜景死惊）
4. **神盘** — 八神（值符蛇阴合虎武地天）

### 三奇六仪
- **三奇**：乙（日奇）、丙（月奇）、丁（星奇）— 最吉之星
- **六仪**：戊（甲子）、己（甲戌）、庚（甲申）、辛（甲午）、壬（甲辰）、癸（甲寅）

### 九宫八卦
坎一(北水)·坤二(西南土)·震三(东木)·巽四(东南木)·中五(中土)
乾六(西北金)·兑七(西金)·艮八(东北土)·离九(南火)

## 算法说明

本 skill 采用**置闰法（拆补法）**进行起局：
- 根据节气确定上中下元 → 定局数
- 根据日干支定旬首 → 定值符
- 根据时干支定值符/值使落宫 → 完成天盘/人盘/神盘排列
- 阳遁顺飞，阴遁逆飞
