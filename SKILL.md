# 奇门遁甲排盘解卦破局 Skill

中国古代最高级术数——**奇门遁甲**，号称"帝王之学"。本 skill 实现时家奇门完整排盘、格局识别、吉凶判断与破局建议。

## 快速开始

### 运行排盘

```bash
python3 scripts/qimen.py -q "用户的问题" [-y 年] [-m 月] [-d 日] [-H 时] [--json]
```

**参数说明：**
- `-q`：求测问题（选填，用于上下文分析）
- `-y/-m/-d`：年月日，不填则用当前时间
- `-H`：时辰（0-23点），不填则用当前时间
- `--json`：JSON格式输出（程序化处理）

**示例：**
```bash
# 当前时间起局
python3 scripts/qimen.py -q "最近事业运如何"

# 指定时间
python3 scripts/qimen.py -q "这笔投资能成吗" -y 2026 -m 4 -d 19 -H 16

# JSON输出（供Agent二次分析）
python3 scripts/qimen.py --json -y 2026 -m 4 -d 19 -H 16
```

## 排盘流程

### 第一步：获取起局信息

从用户提问中提取：
1. **求测问题**：问什么
2. **时间信息**：指定时间或当前时间（奇门强调"此时此刻"）

### 第二步：运行脚本排盘

```bash
python3 scripts/qimen.py -q "问题" [时间参数]
```

### 第三步：解读盘面

1. **看大局**：阳遁/阴遁 + 局数 + 值符值使落宫
2. **看九宫**：各宫的地盘、天盘、八门、九星、八神配置
3. **看格局**：各宫形成的特殊格局（吉格/凶格）
4. **看空亡**：旬中空亡位置
5. **总体评估**：综合吉凶判断

### 第四步：结合问题分析

根据问题类型聚焦相关用神：

| 问题类型 | 用神 | 关注重点 |
|----------|------|----------|
| 事业/工作 | 开门、值符、天心 | 值使落宫、开门落宫状态 |
| 财运/投资 | 生门、戊、天芮 | 生门落宫、戊所在宫 |
| 婚姻/感情 | 休门、乙奇、六合 | 休门落宫、乙奇状态 |
| 疾病/健康 | 天芮（病星）、死门 | 天芮落宫、天芮所临三门五吉 |
| 官非/诉讼 | 惊门、开门、白虎 | 惊门落宫、是否逢凶格 |
| 出行/旅游 | 马星（值符）、腾蛇、玄武 | 所去方向宫位吉凶 |
| 失物/寻人 | 天芮、杜门 | 落宫状态+是否空亡 |
| 求学/考试 | 天辅、丁奇、景门 | 天辅落宫、丁奇状态 |

### 第五步：组织回答

向用户呈现：
1. ⚔️ 局名（阳遁/阴遁N局）+ 时间干支
2. 📋 九宫排盘（文字版洛书布局）
3. 🔍 格局汇总（吉凶分类列出）
4. 📊 总体评估
5. 💡 破局建议（针对问题的具体策略）

## 破局方法论

### 一、总原则

> **吉多进取，凶多固守；吉凶相当，伺机而动。**

### 二、分步破局法

#### Step 1: 定大局
- 数吉格 vs 凶格数量和质量
- 大吉/大凶格有"一票否决权"
- 确定基本策略：攻 / 守 / 观

#### Step 2: 找值符
- 值符是全盘统领
- 值符落宫吉 → 主动出击
- 值符落宫凶 → 宜守不宜攻

#### Step 3: 看三奇
- 乙丙丁三奇是否入墓（入墓则力量大减）
- 三奇是否临吉门吉神
- 三奇是否与用神同宫

#### Step 4: 寻用神
- 根据问题找对应用神宫位
- 分析用神宫位的：
  - 门（吉门/凶门）
  - 神（吉神/凶神）
  - 格（吉格/凶格）
  - 是否空亡（空亡则减力或代表"没有"/"不实"）

#### Step 5: 查生克
- 用神宫与值符宫的五行生克关系
- 用神宫与问事人年命的五行关系
- 生我者吉，克我者凶（一般规律）

#### Step 6: 定应期
- 奇门应期较复杂，主要方法：
  - 值符值使落宫的数字为应期天数/年数
  - 用神宫临马星（值符/腾蛇等）主快
  - 空亡之事务填实之时（冲起空亡）为应期

### 三、常见场景破局模板

#### 问事业
```
看：开门（事业之门）+ 值符（领导）+ 天心（核心）
吉：开门+吉神+吉格 → 升职/顺利
凶：开门+凶神+凶格 → 阻碍/变动
破局：若开门逢凶，找开门的三合宫位是否有救应
```

#### 问财运
```
看：生门（财门）+ 戊（资本）+ 天芮（财物之主）
吉：生门临吉格+三奇 → 财运亨通
凶：生门临死门/凶格 → 破财损耗
破局：找生门的生旺方位行事
```

#### 问感情
```
看：休门（感情门）+ 乙奇（女方）+ 六合（婚姻）
吉：休门+乙奇+六合+吉格 → 良缘
凶：休门+凶神+凶格/空亡 → 波折/不成
破局：看乙奇与庚（男方）的关系
```

#### 问健康
```
看：天芮（病星）+ 死门 + 天芮所落宫
吉：天芮逢空/入墓 → 病轻或无大病
凶：天芮旺相+临凶格 → 病重，需重视
破局：天芮克何宫 → 何脏腑有问题
```

## 注意事项

1. **一事不二占**：同一件事不宜反复起局
2. **时辰精确**：奇门对时辰极敏感，差一个时辰完全不同盘
3. **不妄言生死**：涉及重大凶险时，强调仅供参考
4. **心存正念**：奇门本是兵家韬略，不可用于害人
5. **理性看待**：术数是参考工具，决策还需理性判断
6. **农历注意**：系统自动转换公历为干支，如需农历请特别说明
